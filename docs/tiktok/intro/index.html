说明

温馨提示

如果是初次接入 tiktok 小游戏，在正式接入 tiktok 小游戏之前，建议先查看 常见问题 了解 tiktok APP 本身存在的一些问题，然后再开始正式接入。

注意

目前需要接入 native 版本，h5 版本 tiktok 官方已不再支持。

注意

因为 tiktok 小游戏官方还在建设中，官方文档随时会有更新删改废弃，请保持关注。

  • 添加桌面
