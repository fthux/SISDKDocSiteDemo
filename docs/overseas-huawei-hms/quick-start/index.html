快速开始

  • Unity Building Settings 中平台切换到 Android。

SI 平台设置​

  • 菜单栏将 SI 平台切换到 HMS：SI -> Switch Platform -> HMS。

Huawei HMS Kit 设置​

  • 菜单栏点击 Huawei -> Kit Settings 打开 HMS Kit Settings 窗口：

    • 在 Kits 标签页中选中 Enable Plugins和 Account Kit，如果有内购，也需要选中 In-App Purchases。

    • 在 IAP 标签页中填写或批量上传所有真实的内购商品配置，包含商品 id 和商品类型（消耗型、非消耗型、订阅型）。

TopOn 设置​

  • 菜单栏点击 ThinkupTpnPlugin 打开 Thinkup Tpn Integration Manager 窗口，Select Region 选择海外 Overseas，下方选择需要的 Ad Networkds。
