内购功能

说明​

提示

内购需要登录华为账号，sdk 内部已做处理，应用侧无需关注。

配置内购商品​

在华为开发者后台和 HMS Kit Settings 中配置内购商品。

图

图

购买商品​

提示

支持：消耗型、非消耗型、订阅型，直接传递商品 id 即可。

定义
/// <summary>
/// 购买商品：消耗型、非消耗型、订阅型
/// </summary>
/// <param name="productId">商品 id：支持消耗型、非消耗型、订阅型</param>
/// <param name="price">价格：不参与真实内购，可自行处理</param>
/// <param name="label">商品 label</param>
/// <param name="onPaymentSuccess">HMS 收银台付款回调成功，不表示最终成功</param>
/// <param name="onSuccess">付款成功</param>
/// <param name="onFail">付款失败</param>
/// <returns></returns>
public static IEnumerator PurchaseProduct(
    string productId, float price, string label, Action<string> onPaymentSuccess, Action<string> onSuccess, Action<ErrCode> onFail
)
示例
StartCoroutine(IAP.PurchaseProduct("com.test.xiaohao", 1, "label",  null, (msg) => {
    Debug.Log("购买消耗型商品成功，商品id是：" + msg);
}, (errCode) => {
    Debug.LogError("购买消耗型商品失败，错误码是：" + errCode);
}));

内购恢复​

提示

需要在相关页面（比如设置页面）内添加内购恢复，对消耗型商品进行补单。

消耗型​

定义

/// 恢复消耗型内购
/// </summary>
/// <param name="onSuccess">成功回调</param>
/// <param name="onFail">失败回调</param>
public static void RestoreOwnedConsumableRecords(Action<List<string>> onSuccess, Action onFail)
示例
IAP.RestoreOwnedConsumableRecords(ids => { }, () => { });

获取配置的商品列表​

提示

可通过商品列表来显示经过本地化处理的商品价格。

提示

获取成功后，商品列表就可以通过 SI.productList 访问到，无需重复获取。

定义
/// <summary>
/// 获取配置的商品列表
/// </summary>
/// <param name="onSuccess"></param>
/// <param name="onFail"></param>
public static void FetchAllProducts(Action<List<ProductInfo>> onSuccess, Action<ErrCode> onFail)
示例
IAP.FetchAllProducts(list => {
    for (var i = 0; i < list.Count; i++) {
        Debug.Log($"HMSIAPManager product list {i} {list[i].ProductId}");
    }
}, err => {
    Debug.Log("HMSIAPManager 获取商品列表失败");
});

查询商品是否可用​

消耗型​

不存在该方法。

非消耗型​

定义
/// <summary>
/// 查询非消耗型商品是否可用
/// </summary>
/// <param name="productId">商品 id</param>
/// <param name="callback">成功/失败回调</param>
public static void IsNonConsumablePurchaseAvailable(string productId, Action<bool> callback)
示例
IAP.IsNonConsumablePurchaseAvailable("com.test.feixiaohao", isAvailable => {
    Debug.Log($"[IAP] 非消耗商品是否可用：{isAvailable}");
    stateText.text = "非消耗型商品：" + (isAvailable ? "可用" : "不可用");
});

订阅型​

定义
/// <summary>
/// 查询订阅型商品是否可用
/// </summary>
/// <param name="productId">商品 id</param>
/// <param name="callback">成功/失败回调</param>
public static void IsSubscriptionPurchaseAvailable(string productId, Action<bool> callback)
示例
IAP.IsSubscriptionPurchaseAvailable("com.test.dingyue", isAvailable => {
    Debug.Log($"[IAP] 订阅商品是否可用：{isAvailable}");
    stateText.text = "订阅型商品：" + (isAvailable ? "可用" : "不可用");
});

编辑订阅​

提示

会打开华为手机的原生订阅编辑页面。

管理特定订阅商品​

定义
/// <summary>
/// 打开特定订阅商品编辑页面
/// </summary>
/// <param name="SubscribeProductId">订阅型商品 id</param>
public void RedirectingtoSubscriptionEditingScreen(string SubscribeProductId)
示例
HMSIAPManager.Instance.RedirectingtoSubscriptionEditingScreen("com.test.dingyue");

/// <summary>
/// 打开管理所有订阅商品编辑页面
/// </summary>

HMSIAPManager.Instance.RedirectingtoSubscriptionManagementScreen();
