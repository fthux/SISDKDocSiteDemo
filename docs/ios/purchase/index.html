内购功能

将菜单栏 SI -> Settings -> IosIap 选择为Enable，以开启内购功能。

将IapBackend预制体添加到场景中。

SecretKey: 服务器秘钥。 IsSandbox:沙盒测试时开启，上线时必须关闭。 修改IAPManager中的itemList为苹果商店后台的商品列表。 需要实现IAPManager中的方法：

IAPManager.cs

    /// <summary>
    /// 购买产品
    /// </summary>
    /// <param name="productId">产品ID</param>
    /// <param name="onFailed">失败回调</param>
    /// <param name="onSuccess">成功回调</param>
    public void PurchaseProduct(string _productId, int _price, Action onFailed, Action<string> onSuccess)
    
    
    /// <summary>
    /// 检查用户是否有需要处理的退款
    /// </summary>
    /// <param name="onComplete">退款检查完成回调，参数为是否有待退款项目及退款产品订单列表</param>
    public void CheckAndProcessRefunds(Action<bool, List<IapBackend.BriefOrder>> onComplete = null)
    
    
    /// <summary>
    /// 检查用户是否有需要处理的已付款但尚未发货订单，（ProcessPurchase方法在每次初始化后会自动运行，并获取所有已付款订单，建议在ProcessPurchase中处理未生效订单）
    /// </summary>
    /// <param name="onComplete">退款检查完成回调，参数为是否有待退款项目及退款产品订单列表</param>
    public void GetOrdersPaidButNotDelivered(Action<bool, List<IapBackend.BriefOrder>> onComplete = null)
